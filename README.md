# Annai Savvy AI

## Version

v0.5

## Project Description

Annai Savvy AI is an AI-powered Business Intelligence platform that automatically analyzes Excel workbooks, detects datasets, generates KPIs, recommends charts, builds dashboards, and will later provide AI-generated business insights.

## Current Features

- Excel Upload
- Multi-sheet Processing
- Dataset Identification
- Relationship Detection
- KPI Generation
- Chart Recommendation
- Dashboard Generation

## Backend Technology

- Python 3
- FastAPI
- Pandas

## Planned Features

- React Dashboard
- AI Chat Assistant
- PDF Reports
- Power BI Export
- Database Connectivity
- User Authentication
